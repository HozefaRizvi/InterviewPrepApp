Interview Preparation Application Run:

First Start Python Flask Backend (NewBackend):
	- Go to app.py
	- Press CTRL+F5 then debug will start in terminal
	- Copy the link of the url 
Now Start the front End Application (interview_prep_app):
	- Make New Terminal 
	- Install Node Modules : npm install
	- Run following command : npx expo start 
	- Pase the Link url copied from backend to api.js
